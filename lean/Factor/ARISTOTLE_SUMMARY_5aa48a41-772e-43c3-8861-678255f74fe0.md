## The Three Questions Answered

### 🧭 The Heuristic Compass
- Defined the **angular cross-product** `cross(P₁,P₂) = a₁b₂ - b₁a₂` as an exact integer-valued distance metric
- Proved the **Angular Pythagorean Identity**: `cross² + dot² = (c₁c₂)²`
- The **compass reading** `n/m = tan(θ/2)` gives a natural ordering on Berggren nodes, isomorphic to the Stern–Brocot tree
- Proved compass reading always stays in `(0, 1)` — the missile is always on the correct hemisphere
- **Key discovery**: M₃ reading < M₂ reading always (proved), but M₂ does NOT always decrease the reading (disproved!) — this means M₁ is essential for full coverage

### 🔄 Path Correction
- Formalized parent descent as exact course correction using the inverse Berggren matrices
- Computationally verified parent-child roundtrips for all three branches
- The correction is **algebraically exact** (no floating-point error accumulation)
- Parent computation is equivalent to the Euclidean algorithm on Euclid parameters

### 🎯 Target Acquisition
- Connected Pythagorean triples to Gaussian integers via norm multiplicativity (proved: `N(z₁z₂) = N(z₁)N(z₂)`)
- Demonstrated **factoring by Berggren tree search**: BFS finds triples with `c | N`, directly yielding factors
- Example: factoring 65 = 5×13 finds both prime factors; factoring 85 = 5×17 likewise
- Proved `a < c` for positive Pythagorean triples (ensuring nontrivial representations are distinguishable)
