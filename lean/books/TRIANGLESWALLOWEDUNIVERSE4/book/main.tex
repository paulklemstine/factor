\chapter*{Note on the Formal Proofs}
\addcontentsline{toc}{chapter}{Note on the Formal Proofs}
\markboth{Formal Proofs}{Formal Proofs}

The following appendices contain the complete Lean~4 source code for the formal 
proofs referenced throughout this book. These proofs have been machine-verified 
using the Lean~4 theorem prover with the Mathlib library, ensuring mathematical 
certainty beyond what informal argument alone can provide.

Each appendix corresponds to a chapter of the book. The proofs formalize the key 
theorems, establishing with absolute rigor the mathematical claims made in the 
narrative. The reader interested in verification may compile these files using 
Lean~4 (version 4.x) with an appropriate Mathlib installation.

\bigskip
\noindent\textit{Notation guide:} In Lean~4, \texttt{theorem} and \texttt{lemma} 
introduce proven statements, \texttt{def} introduces definitions, and \texttt{sorry} 
marks statements that are asserted but not yet proven. The symbol \texttt{:=} 
separates a declaration from its proof term, while \texttt{by} introduces a 
tactic-mode proof.
