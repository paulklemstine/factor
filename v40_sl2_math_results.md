# v40: SL(2) Consequences of Berggren mod p

## Summary

Eight experiments exploring the group-theoretic, modular, and arithmetic
consequences of the Berggren Pythagorean triple generators.

## Main Theorems

### T1: Berggren Group = Theta Group
The Berggren generators M1=[[2,-1],[1,0]] and M3=[[1,2],[0,1]] (det=1 part)
generate the **theta group** Gamma_theta = <S, T^2>, which has **index 3** in SL(2,Z).

Key identity: **S = M3^{-1} * M1**.  Verification:
```
[[1,-2],[0,1]] * [[2,-1],[1,0]] = [[0,-1],[1,0]] = S
```
And T^2 = M3.  So <M1,M3> = <S,T^2> = Gamma_theta.

### T2: Surjectivity onto SL(2,F_p) for all odd primes
For all odd primes p, <M1,M3> mod p = SL(2,F_p).  Verified through p=31.

**Proof**: Since p is odd, 2 is invertible in F_p, so T = (T^2)^{2^{-1}} is
in <S, T^2> mod p.  Then S and T generate SL(2,F_p).

**Failure at p=2**: T^2 = I mod 2, so <S,T^2> mod 2 = {I, S}, order 2,
vs |SL(2,F_2)| = 6.  This is consistent with index 3.

### T3: Theta Function Connection
The canonical modular form for Gamma_theta is the **Jacobi theta function**
theta(tau) = sum q^{n^2} (weight 1/2 with multiplier).

theta^4 = sum r_4(n) q^n counts representations as sum of 4 squares.
Jacobi's formula r_4(n) = 8 * sum_{d|n, 4 nmid d} d verified for n <= 50.

L-function: L(s, theta^4) = zeta(s) * zeta(s-1) (Eisenstein, not cusp form).
Hecke eigenvalue lambda(p) = p+1 verified for all odd p <= 23.

### T4: Character Variety
Character variety point: (tr(S), tr(T^2), tr(ST^2)) = (0, 2, 2).
Fricke surface value: x^2 + y^2 + z^2 - xyz = 8.
Commutator [S, T^2] has trace 6, eigenvalues 3 +/- 2*sqrt(2) (hyperbolic).

### T5: Expander Graphs (Bourgain-Gamburd)
The Cayley graphs Cay(SL(2,F_p), {M1^{+/-1}, M3^{+/-1}}) form an expander
family by Bourgain-Gamburd (generators surject onto SL(2,F_p) for all odd p).

### T6: ADE Tower
- mod 3: SL(2,F_3) = binary tetrahedral (order 24) -> E_6 (McKay)
- mod 5: SL(2,F_5) = binary icosahedral (order 120) -> E_8 (McKay)
- mod 7: SL(2,F_7), PSL(2,F_7) = Klein quartic group (order 168)

### T7: Integer Arithmetic
All entries of Berggren products are integers (Z, not Z[1/2]).
Max entry grows exponentially: ~2^depth (spectral radius ~ golden ratio).

### T8: Connection to X_0(4)
Gamma_theta is conjugate to Gamma_0(2) via T-conjugation.
PPTs correspond to rational points of X_0(4).
